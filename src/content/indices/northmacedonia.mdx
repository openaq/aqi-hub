## Background

The AQI in North Macedonia is designed to communicate the short-term (hourly) health implications of air pollution in a standardized format. North Macedonia’s system (via the Ministry of Environment and Physical Planning) is not an independent methodology. It is a national implementation of the European Environment Agency's European Air Quality Index framework.

## Methods

The AQI is calculated using hourly concentration data from five key pollutants: PM2.5, PM10, Nitrogen Dioxide (NO2), Ozone (O3), and Sulfur Dioxide (SO2). All pollutant concentrations are expressed in micrograms per cubic meter (µg/m³).

For each monitoring station (or modelled grid cell), the most recent hourly concentration for each of the five pollutants is obtained. 

A station must report required pollutants (NO₂ and at least one particulate pollutant; O₃ required for background/industrial stations) for a full AQI classification. If real-time data are missing for NO2, PM2.5, PM10, or O3, values may be gap-filled using downscaled Copernicus Atmospheric Monitoring Service (CAMS) model data: 
Difference method (for NO2, PM2.5, PM10): The CAMS model value is adjusted by adding or subtracting the average historical difference between measured and modelled values for the same hour over at least three of the previous four days. 
Multiplicative method (for O3): The CAMS model value is corrected using the average ratio between previously measured and modelled values for the same hour over at least three of the previous four days. 
No gap-filling is performed for SO2

Each pollutant's hourly concentration is compared against predefined concentrations corresponding to the six qualitative air quality categories: Good, Fair, Moderate, Poor, Very Poor, and Extremely Poor.

Unlike breakpoint-based systems (such as the U.S. AQI), the European/North Macedonian AQI does not use linear interpolation to calculate a numeric sub-index. Instead, pollutant concentrations are directly classified. 

The overall AQI level for a monitoring station is determined by identifying the pollutant with the worst (highest severity) category.
